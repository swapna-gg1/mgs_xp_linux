Legato Graphics Composer for Linux

This binary is the Linux version of the Legato Graphics Composer.  Required
dependencies must be installed before it can be run.

Dependencies:

FreeType2
FreeImage

These libraries can be installed through your system's package manager or
they may be included with Qt itself

Please see "https://github.com/Microchip-MPLAB-Harmony/gfx/wiki" for 
additional development resources.
