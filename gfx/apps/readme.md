# Microchip Graphics Suite (MGS) Harmony Application Support

Microchip Graphics Suite (MGS) Harmony provides various application examples for a number of device families. 

For feature-rich, professional Microchip Graphics Suite demonstrations and demonstrations from third-party graphics libaries, refer to user guides on [Microchip Developer Help](https://developerhelp.microchip.com/xwiki/bin/view/software-tools/mgs/dev-kits) or search for "Microchip Graphcis Suite" on [MPLABÂ® Discover](https://mplab-discover.microchip.com/v2?dsl=Microchip+AND+Graphics+AND+Suite).
